1. Prepare
1.1. Create new auth token and save password
1.2. Create a new credential on BOTH databases.
BEGIN
  DBMS_CLOUD.CREATE_CREDENTIAL(
    credential_name => '<CRED_NAME>',
    username => 'oracleidentitycloudservice/<user>@gmail.com',
    password => '<auth token password>'
  );
END;
/

1.3. Create Bucket in Object Storage
https://objectstorage.us-ashburn-1.oraclecloud.com/n/<namespace>/b/<bucket-name>/o/
Save namespace and bucket-name.
1.4. Make it public

2. expdp
expdp usr/pwd@db credential=<CRED_NAME> dumpfile=https://objectstorage.us-ashburn-1.oraclecloud.com/n/<namespace>/b/<bucket-name>/o/file_dmp.dmp SCHEMAS=<> logfile=logfilw.log directory=data_pump_dir parallel=4

3. impdp
impdp usr/pwd@db credential=<CRED_NAME> dumpfile=https://objectstorage.us-ashburn-1.oraclecloud.com/n/<namespace>/b/<bucket-name>/o/file_dmp.dmp

4. Public synonims
Create all public synonims
Re-commpile all PL/SQL code

5. Clean up
5.1. Credentials
BEGIN
   DBMS_CLOUD.DROP_CREDENTIAL('<CRED_NAME>');
END;
/

5.2. Delete log files.
BEGIN
  DBMS_CLOUD.DELETE_FILE ('DATA_PUMP_DIR','<name of file to delete>');
END;
/
